# Service Manager

A comprehensive solution for managing vehicle service operations, customer details, and inventory.

## Project Structure

- **frontend/**: Contains the main Flutter application source code.

## Getting Started

To run the application locally, follow these steps:

1.  **Navigate to the project directory:**
    ```bash
    cd frontend
    ```

2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the application:**
    ```bash
    flutter run
    ```
